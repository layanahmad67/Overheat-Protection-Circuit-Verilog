# Overheat-Protection-Circuit-Verilog
Overheat Protection Circuit

Overview

This project presents a digital Overheat Protection Circuit designed to protect electronic devices from excessive temperatures. The system continuously monitors temperature values and activates a cooling fan or alarm when predefined thresholds are reached.

Features

Temperature Monitoring
Fan Control
Alarm Activation
Multi-Level Temperature Thresholds
Verilog Implementation
Active-HDL Simulation

Technologies

Verilog HDL
Active-HDL
Digital Logic Design
